./data - общий каталог, через который инструменты проекта обмениваются данными (монтируется как том в контейнеры).

```
data/
├── certs/   # сертификаты и приватные ключи (PEM/.cer/.key)
│             #   - cpcert записывает сюда ключи из контейнеров
│             #   - xades-signer читает отсюда cert/key для подписи
└── xml/      # вход/выход XML для подписи xades-signer
```
