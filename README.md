# Crypto-Любитель

Если вы программист и вам поставили задачу вроде интеграции OAuth ЕСИА, подписания каких-либо данных ГОСТ-алгоритмами, то вы наверняка столкнулись с набором программ КриптоПро и нити судьбы привели вас сюда. Данный репозиторий это набор инструментов для работы с ГОСТ-криптографией без КриптоПро и конечно же не претендует на роль профессионального инструмента - ведь он уже есть, [десятки](https://cryptopro.ru/products) программ, библиотек, форум и документация - потратите недели, чтобы разобраться во всех нюансах. Мне в моей работе уже нужно было реализовывать подпись данных CAdES-BES, усиливать CAdES-BES подписи до CAdES-T, подписывать XML пакеты XAdES-BES/XAdES-T, валидировать подписи, интегрировать авторизацию через ЕСИА и каждый раз я осознавал недостаток профессионализма при работе с КриптоПро - то pfx контейнеры, с которыми не работает ни одна библиотека, которую мне доводилось видеть, то 3 утилиты для подписания разными форматами и требующие ключи и тяжело работающие с контейнерами.

Основная задача - обойтись без проприетарного софта и решить задачи, используя в основном OpenSSL 1.1.1 + gost-engine + библиотеки для работы с ГОСТ, чтобы потом это легко интегрировалось с современными инструментами и библиотеками, используемыми в веб-разработке. Все работает в контейнерах, поэтому нет нужды устанавливать на свою машину довольно старую версию OpenSSL и прочие инструменты.
